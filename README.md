# 宝宝体温与给药记录 App

这是一个基于 `Kotlin + Jetpack Compose` 的安卓原生项目骨架，围绕以下第一版目标搭建：

- 记录体温
- 记录给药
- 查看今日概览
- 查看体温趋势图
- 标记给药时间点
- 导出当前概览为图片

## 当前状态

当前项目已经包含：

- 基础 Gradle 工程结构
- Compose 主题与首页视觉方向
- 首页 / 记录 / 趋势 / 设置 四个主页面
- Room 本地数据库与 Repository
- 体温折线图、正常范围带、给药点位标记
- AlarmManager + BroadcastReceiver 提醒链路
- Android 13 通知权限申请
- 导出当前界面为 PNG 图片到系统相册

当前项目还未接入：

- 多宝宝支持
- 真正的图片分享流程
- 提醒恢复策略（开机后重建、提醒历史）
- 医疗参考范围配置来源确认

## 本机启动前提

当前工作机还没有可用的 `Java` 与 `Gradle` 环境，所以本次先完成了工程骨架与代码落地，未能实际编译验证。

建议你在 Android Studio 中执行：

1. 安装 JDK 17
2. 用 Android Studio 打开 `baby-care-app`
3. 让 IDE 自动补齐 Gradle Wrapper
4. 同步项目并运行

建议下一阶段优先做：

1. 增加提醒记录表，支持开机后自动恢复未完成提醒
